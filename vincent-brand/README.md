## 核心能力 / Core Skill

**知識體驗設計 Knowledge Experience Design**

不只是內容，也不只是設計——而是兩者加上「教學邏輯」的合成。

知識如何被遇見：以什麼順序、什麼節奏、什麼視覺邏輯、通過什麼互動方式呈現，才能讓人真正產生認知轉變。

> Information doesn't change people. Experience does.

## 服務範疇 / What I Do

### 工作坊與簡報設計
將演講者的想法轉化為有結構、逐步揭示的投影片體驗。  
設計「知識流」——哪個概念先出現，哪個後出現，都有原因。  
建構互動式 HTML 簡報，而不是 PowerPoint。

### 知識頁面設計
單頁網站，作為方法論展示、課程著陸頁或思想領導者的品牌中心。  
結構像工作坊：前提 → 框架 → 論據 → 應用 → CTA。  
頁面本身就是能力的證明。

### 課程架構設計
協助教育者、教練與知識工作者，把自己的專業組織成可教授、可分享的框架。  
找出「最小可行框架」——你的工作背後真正的核心心智模型。

### 互動內容系統
循序揭示邏輯、摺疊式深度架構、視覺化框架（DFV、Octalysis、Hormozi 公式）渲染為互動元件。  
同時適用於現場簡報與非同步自學。

## 常用框架 / Frameworks

| 框架 | 應用情境 |
|---|---|
| DFV (欲求性 / 可行性 / 存續性) | 創新策略、產品定位、個人發展 |
| MECE 思維 | 問題拆解、課程架構 |
| Hormozi 價值公式 | 產品設計、定價策略、價值主張 |
| Octalysis（周郁凱） | 參與設計、遊戲化邏輯 |
| 循序揭示 | 簡報節奏、學習體驗 |
| ICP × HXC | 受眾定位、人物設計 |
